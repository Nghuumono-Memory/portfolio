Concept · Functions
Why Functions Make Code Better
A function is a reusable block of code that performs a specific task. Instead of writing the same logic repeatedly, you define it once and call it anywhere. In CorroCheck, we used functions to handle corrosion calculations and Firebase data fetching.

A simple Python example:
def calculate_corrosion_rate(mass_loss, area, time):
    rate = mass_loss / (area × time)
    return rate
The corrosion rate formula used in mining is: CR = (K × W) / (A × T × D) where W = mass loss, A = area, T = time, D = density.
Concept · Variables & Data
Variables, Data Types & Why They Matter
Every piece of information in a program is stored in a variable. The data type determines what operations you can perform. In CorroCheck, we store corrosion readings as floats, location names as strings, and report dates as datetime objects.

reading = 0.45         # float
location = "Shaft 3"    # string
flagged = True          # boolean
Choosing the right data type prevents bugs and makes code efficient — a key lesson from both Python class and the MATLAB Onramp course.
